Experiment 1 - PRCG DND Item-Maker

Description

This project aims to be a useful tool for Dungeon Masters playing DND 5E, by allowing them to procedurally generate random loot to fill their shops.

Technical

For this project I created new a new template for the procedurally generated text, imitating the project "bad fantasy quests" given to us.
I used new fantasy names and weapon types, and imitated the silly, lighthearted tone of the previous project.
After doing this, I attempted to integrate the existing weapon scheme of DND, to make the tool more applicable to in-game use.
I did this by incorporating integers into the template, and using them to relate back to the weapon damages, types, properties, and ranges of dnd.
Finally, I innovated upon this project by storing and using the data created from this procedurally generated template.
I stored the results of each proc gen call into a list, and iterated through the list to find the integers.
I ran a simple calculation to determine the cost of the item based on the number of dice, the type of dice, or it's attack range.
This displays as the cost at the bottom of the page.

Reflection

This project was relatively simple.
The process was smooth, and was planned out from the very beginning.
The main difficuly with this extension of the original premise was modifying the original code to store the data created by the procedural generator
Additionally, I couldn't get the newlines to work on the webpage, even though glitch handles it fine. I imagine that I would have to separate the template into different divs in html?
Despite these challenges, I am happy with this final product, and am proud of the unique way it expands upon the original goals of the assignment.
